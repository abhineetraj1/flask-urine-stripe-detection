# Urine strip colour detection
I've built a urine test strip analyzer. Users can upload a picture of a test strip (like the ones in the sample folder), and after submitting the form, the system will identify and display the colors of the 10 test areas on the strip, using opencv-python.

## Installation
*	Download python3.10
*	Install required libraries
```
pip3 install -r requirements.txt
```
## Execution
*	Download the source code:-
```
git clone https://github.com/abhineetraj1/flask-web-hosting
cd flask-web-hosting
```
*	Open terminal and run flask server
```
flask run -h localhost -p 5000
```
*	Open your default browser and navigate to localhost:5000 in that browser
